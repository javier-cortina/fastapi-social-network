alembic
	inside the (venv) command line, type:
	alembic init --help
	alembic init alembic  # creates alembic directory
	
	alembic revision --help
	alembic revision -m "create posts table"  # "commits" change and creates file within /versions. 
	# We need to define the upgrade and downgrade the funcions. We must do this according to the documentation under https://alembic.sqlalchemy.org/en/latest/api/ddl.html
	
	alembic upgrade d1da4814ce19
	# or
	alembic upgrade head
	
	# type this in order for alembic to automatically interpret the database model from the models.py file:
	alembic revision --autogenerate -m "auto generate the database"
